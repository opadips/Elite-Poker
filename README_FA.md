## ✨ ویژگی‌ها
### 🏠 سیستم لابی
- **لیست لابی** – بگردید بین میزهای موجود با تعداد بازیکن، بالاترین امتیاز، حالت بازی و وضعیت قفل.
- **کارت‌های لابی زنده** – حالا توی همون کارت لابی تعداد بازیکنای فعال، blindها، چیپ شروع و اگه دستی در جریان باشه اندازه پات و راند بازی رو هم میبینی.
- **ایجاد میز خصوصی/عمومی** – اسم، توضیحات، رمز عبور، میزان بلیز، چیپ اولیه (تا 1,000,000)، و حالت بازی (مسابقات / نقدی) رو تنظیم کن.
- **چت عمومی** – قبل از رفتن به میز، با همه بازیکنان آنلاین چت کن.
- **پنل بازیکنان آنلاین** – ببین کی توی لابی آنلاینه.
- **صف انتظار (Waitlist)** – اگه یه میز پر بود (حداکثر 10 نفر)، برو توی صف؛ وقتی جا باز شد خودت میای توی بازی.
- **کنترل‌های ادمین** – فقط سازنده میز می‌تونه لابی رو ریست کنه یا بازیکن‌ها رو کیک کنه.
- **کیک کردن بازیکن** – ادمین با دکمه ❌ کنار اسمشون میتونه بازیکن‌ها رو بیرون کنه.
- **بازگشت به لابی** – از میز خارج شو و بدون قطع شدن برگرد به لیست لابی.

### 🎤 یه دیلر باحال و کنایه‌گو
- یه دیلر طناز و بامزه تمام حرکتای بازی (فولد، کال، رایز، آل‌این و ...) رو با بیش از ۱۰۰ خط مختلف روایت می‌کنه.
- پیامای دیلر فوری بالای صفحه میان و بعد از یه لحظه محو میشن—تمیز و بدون شلوغی.

### 🎮 گیم‌پلی اصلی
- قوانین کامل تگزاس هولدم (preflop, flop, turn, river, blinds, side pots).
- چندنفره Real‑time از طریق **WebSocket** (2 تا 10 بازیکن در هر میز).
- **حالت تماشاچی** – بازی رو نگاه کن و با چیپای شروع هر موقع خواستی بشین سر میز.
- **شرط‌بندی جانبی (Side Bet)** – بازیکنایی که فولد دادن میتونن روی بازیکنای فعال شرط ببندن (50% سود). تو حالت دو نفره که حریف فولد بده، پول شرط برگشت داده میشه—خبری از سواستفاده نیست.
- **حالت مسابقه‌ای (Tournament)** – چیپا فقط وقتی یه نفر مونده ریست میشن؛ برنده یه امتیاز میگیره و همه با چیپ تازه شروع میکنن.
- **حالت نقدی (Cash)** – چیپا همینجوری جمع میشن و ریست اجباری نداریم.
- **تایمر خودکار** – ۲۰ ثانیه وقت داری، وگرنه اگه پولی نداشته باشی خودکار چک میشی، در غیر این صورت فولد. دور کارت بازیکن فعال یه **حلقه نئونی شیک** میاد که رنگش از آبی → سبز → نارنجی → قرمز عوض میشه.
- **مکث / ادامه** – هر وقت خواستی بازی رو نگه دار (تایمر فولد متوقف میشه) و بعد ادامه بده.

### 🎨 ظاهر و تجربه کاربری
- **۶ تم خوشگل و چشمنواز** با CSS خالص (بدون فایل خارجی):
  - `Classic` · `Royal Crimson` · `Emerald Luxe` · `Sapphire Noir` · `Neon Jungle` · `Void Pulse`
- **شخصی‌سازی پشت کارت** – 6 طرح آماده که با تم‌ها ست میشن، ذخیره توی localStorage.
- **نمای صندلی** – ثابت (صندلی تو همیشه پایینه) یا چرخشی، از توی تنظیمات عوضش کن.
- **انیمیشن‌های باحال**:
  - اسپین 3D برای کارت‌ای مشترک
  - حرکت چیپ‌ها با قوس پارابولیک
  - متن دست برنده با فونت و رنگ مخصوص هر تم
  - نمایش با تأخیر کارت‌ای آل‑این (۲ ثانیه بین هر خیابون)
- **نشونگرهای بصری** – تاج ادمین فقط روی صندلی خودش، برچسب SB/BB، برچسب Dealer که تا شروع اولین دست قایم میشه.

### 🔊 صداها (Web Audio API)
- **بدون فایل صوتی** – همش با Web Audio API ساخته میشه.
- صدای پخش کارت، کلیک چیپ، بوق تایمر، فانفار برنده، و یه آهنگ هیجانی برای آل‑این.
- دکمه خاموش/روشن صدا هم که هست :)

### 📊 Noob Mode (من نوب سگم)
- وقتی فعالش کنی، یه **نوار احتمال زنده** میاد که شانس بردن دست رو نشون میده (شبیه‌سازی مونت کارلو با ۲۰۰۰ بار اجرا).
- رنگ نوار: سبز (احتمال کم) → نارنجی (متوسط) → قرمز (زیاد).

### 🏆 دستاوردها (Achievements)
- فقط لحظه‌های واقعاً خاص و کمیاب دستاورد باز میکنن—دیگه خبری از اسپم توی چت نیست.
- مثلاً بردن با Royal Flush، بردن یه پات میلیونی، ۱۰ برد پشت سر هم، یا بردن یه دست در حالی که فقط ۱ چیپ برات مونده.

### 💬 چت و Quick Chat
- چت میز با تاریخچه موندگار.
- **حباب‌های گفتار** چند ثانیه بالای سر فرستنده ظاهر میشن.
- **پنل Quick Chat** – با یه کلیک ایموجی پیام‌های آماده بفرست (احساسات، بازی، احوالپرسی).
- **پیام خصوصی** – بنویس `/w اسم‌بازیکن پیامت` تا مخفیانه چت کنی (رنگ بنفش، فقط طرفین میبینن).
- چت بعد از چند ثانیه بی‌استفاده بودن خودش جمع میشه و همیشه روی همه المان‌ها سواره.

### 👤 آمار و تاریخچه بازیکن
- لیدربرد با کلیک روی اسم‌ها باز میشه: دستای بازی‌شده، پاتای برده، باختا، بزرگترین پات، بهترین دست، نرخ برد.
- **تاریخچه دست‌ها** – دورای اخیر رو از منوی تنظیمات ببین.

### 🛡️ بهبودهای دیگه
- وقتی بازیکنی که نوبتشه کانکشنش قطع بشه، خودکار فولد میشه—خداحافظ قفل شدن میز.
- لاگ JSON ساختاریافته روی سرور برای دیباگ زنده.
- چت همیشه بالای چیپ‌ها و صندلی‌ها و کارت‌ها میمونه.

## 🚀 راه شروع سریع
### پیش‌نیازها
- Node.js ≥ 16
- npm یا yarn
### نصب
```bash
git clone https://github.com/opadips/Elite-Poker.git
cd Elite-Poker
```
**بک‌اند (سرور WebSocket)**
```bash
cd backend
npm install   # یا yarn
node server.js
```
سرور روی `ws://0.0.0.0:3000` اجرا میشه.
**فرانت‌اند (React + Vite)**
```bash
cd ../frontend
npm install
npm run dev
```
برو به `http://localhost:5173` (یا آدرس LAN خودت) و بازی کن.

## 🕹️ نحوه بازی

### ۱. ورود و پیوستن
- یه **اسم کاربری** (حداکثر ۱۵ کاراکتر) بزن و **Enter Lobby** رو بزن.
- لیست لابی رو ببین یا یه **میز جدید بساز**.
  - اسم میز، رمز عبور (اگه خواستی)، blindها، چیپ شروع (تا ۱ میلیون)، و حالت بازی رو تنظیم کن.
- روی یه میز کلیک کن تا واردشی. اگه رمز داشت، بزنش.

### ۲. شروع بازی
- اول به عنوان **تماشاچی** وارد میشی – **Sit In** رو بزن (پایین راست) تا بشینی پای میز.
- وقتی توی دست نیستی، دکمه **Ready** رو بزن تا اعلام آمادگی کنی.
  - وقتی آماده باشی، دور صندلیت سبز روشن میشه.
  - وقتی همه آماده بودن، اولین دست خودکار استارت میخوره.

### ۳. حرکت کن
- تو نوبت خودت، از دکمه‌های اکشن (Fold, Check, Call, Raise, All‑in) استفاده کن.
- بازیکنایی که فولدن میتونن رو بازیکنای فعال **شرط جانبی** ببندن.

### ۴. چت و تنظیمات
- از منوی **⚙️ Settings** تم، پشت کارت، نمای صندلی، صدا، Noob Mode، مکث، تاریخچه و برگشت به لابی رو عوض کن.
- توی چت بنویس، از Quick Chat ایموجی بفرست، یا با `/w اسم پیام` مخفیانه چت کن.

### ۵. کنترل ادمین
- سازنده میز میتونه لابی رو ریست کنه یا بازیکن‌ها رو بیرون کنه.
